## Учебный проект по вычислительным методам

###  Задание 
Сравнить противопоточной схемы и схемы Бима-Уорминга для решения краевой задачи для уравнения переноса

###  Решение 
[Beam-Warming Scheme](Beam-WarmingScheme.ipynb)

### Ход выполнения
Решаем первую краевую задачу для уравнения переноса в виде:

$$
\begin{cases}   
\dfrac{\partial u}{\partial t} + a\dfrac{\partial u}{\partial x} = 0,  \\
u(0, t) = 1,   \\
u(1, t) = 0,   \\
u(x, 0) = 0.   
\end{cases}  
$$

$$  
0 < t < 1,  0 < x < 1, a = 1   
$$

Используем противопоточную схему для a > 0:  

$$   
\dfrac{u_j^{n+1} - u_j^n}{\tau} + a \dfrac{u_j^{n} - u_{j-1}^n}{h} = f(n),       
j = 1..N  
$$    

А также схему Бима-Уорминга: 

$$   
u_j^{n+1} = u_j^{n} - \delta(u_j^{n} - u_{j-1}^n) + \dfrac{\delta}{2} (u_j^{n} - 2u_{j-1}^n + u_{j-2}^n),    
\delta =  \dfrac{a \tau}{h}
$$ 

Результаты расчета представляем в виде графиков.

Полученные результаты сравниваем с точным решением:  

$$
u(x,t) = 
\begin{cases}   
1,  0 < x < t, \\
0,  t < x < 1.
\end{cases}  
$$
